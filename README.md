# Algo_901-algorithme_genetique
Implémentation d'un algorithme génétique en POO.

Le projet est séparé en deux dossiers. Le premier "core" regroupe toutes les classes permettant de créer une population tandis que le second "operators" regroupe toutes les classes permettant d'intéragir avec la popualtion.  

Pour plus de détail, "core" contient: 
- AlgorithmeGenetique.py
- Coordonnees.py
- Individu.py
- Population.py

"operators" contient: 
- Codage.py
- Crossover.py
- Mutation.py
- Selection.py
- MantisseExposant.py
