# Algo_901-algorithme_genetique
Ce projet propose une implémentation orientée objet d’un **algorithme génétique**, permettant de manipuler une population d’individus, d’appliquer des opérateurs génétiques et de simuler une évolution.

L’architecture est organisée en deux dossiers principaux :

Le projet est séparé en deux dossiers. Le premier "core" regroupe toutes les classes permettant de créer une population tandis que le second "operators" regroupe toutes les classes permettant d'intéragir avec la popualtion.  

Pour plus de détail, "core" contient: 
- AlgorithmeGenetique.py
- Coordonnees.py
- Individu.py
- Population.py

"operators" contient: 
- Codage.py
- Crossover.py
- Mutation.py
- Selection.py
- MantisseExposant.py

La classe AlgorithmeGenetique permet de regrouper toutes les classes et de simuler un exemple d'utilisation du projet.
